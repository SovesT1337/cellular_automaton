# Описание тестов

## Обзор

Набор из 25 разнообразных тестов, охватывающих различные типы графов и топологий.  
**Общее время выполнения**: ~5 минут

## Метрики производительности

### Ожидаемые результаты

**CPU**:
- Малые графы (<10K): ~100-200 млн ячеек/с
- Средние графы (10K-100K): ~80-120 млн ячеек/с
- Большие графы (>1M): ~50-100 млн ячеек/с

**CUDA (RTX 3060)**:
- Малые графы: медленнее CPU (накладные расходы)
- Средние графы: 2-5x ускорение
- Большие графы: 10-20x ускорение

### Время выполнения (прогноз)

| Категория | Тестов | CPU время | CUDA время | Итого |
|-----------|--------|-----------|------------|-------|
| Случайные графы | 5 | ~50 сек | ~5 сек | ~55 сек |
| 1D решетки | 2 | ~20 сек | ~3 сек | ~23 сек |
| 2D сетки | 4 | ~45 сек | ~8 сек | ~53 сек |
| 3D кубы | 3 | ~60 сек | ~12 сек | ~72 сек |
| Высокоразмерные | 4 | ~40 сек | ~8 сек | ~48 сек |
| Крупномасштабные | 7 | ~120 сек | ~20 сек | ~140 сек |
| **ИТОГО** | **25** | **~335 сек** | **~56 сек** | **~391 сек (~6.5 мин)** |

*Примечание: время включает оба вычисления (CPU + CUDA) для каждого теста + накладные расходы*

## Функции обратной связи

### XOR (60% тестов)
```cpp
result = neighbor_1 XOR neighbor_2 XOR ... XOR neighbor_N
```
- **Линейная** функция
- **Обратимая** (детерминистская)
- Применение: LFSR, криптография

### Majority (40% тестов)
```cpp
result = (count(neighbors == 1) > N/2) ? 1 : 0
```
- **Нелинейная** функция
- **Необратимая** (информация теряется)
- Применение: консенсус, сглаживание, Game of Life

## Выходная последовательность

Для каждого теста формируется выходная последовательность:
- Извлекаются значения из первых 5 ячеек
- Каждые N шагов (N = steps/10)
- Результат: битовая последовательность длиной ~55 бит

**Проверка корректности**: выходные последовательности CPU и CUDA должны совпадать на 100%.


## Анализ результатов

После завершения программа выводит:

1. **Детальные результаты** каждого теста
2. **Сводную таблицу** со временем CPU/CUDA и ускорением
3. **ASCII график** сравнения производительности
4. **Выводы** о эффективности CUDA для разных типов графов
5. **Общую статистику** времени выполнения
